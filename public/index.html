Configure your OpenRouter API key to access various AI models from multiple providers

What is OpenRouter?

OpenRouter is a unified API that provides access to hundreds of AI models from various providers including OpenAI, Anthropic, Google, Meta, and many others. With a single API key, you can access models like GPT-4, Claude 3.5, Gemini Pro, and more.

Available Models

  • OpenAI: GPT-4, GPT-4o, GPT-3.5-turbo
  • Anthropic: Claude 3.5 Sonnet, Claude 3 Haiku
  • Google: Gemini Pro, Gemini Flash
  • Meta: Llama 3.1, Code Llama
  • And many more from various providers

Features

  • Secure API key storage with encryption
  • Automatic key validation
  • Masked key display for security
  • Easy key management and removal
  • User-scoped access control
  • Integration with BrainDrive chat system
Important: This plugin requires an OpenRouter API key. You can get one for free at https://openrouter.ai/keys

How to Use

  1. Get your OpenRouter API key from OpenRouter Platform
  2. Install this plugin in BrainDrive
  3. Enter your API key in the settings
  4. Start using various AI models in the chat module
